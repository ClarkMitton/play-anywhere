Immersive Learning · Design Your Own Lesson

What you can build

A lesson is a timeline of slots. Each slot drives three screens at once — you just fill the three boxes.
Host · TeacherWhat you show & control at the front — the question, the media, the buttons.
Touch Screen 1What learners see and tap.
Touch Screen 2What learners see and tap (often mirrors Screen 1).

Show something

  • Text slide — big text + subtitle
  • Image — with optional title
  • YouTube — paste the link
  • Embed — any website
  • Standby — calm holding screen

Make it interactive

  • Confidence checker — rate 1–10, 😡→😄, or agree↔disagree; live chart + average
  • Question — multiple choice · true/false (live results on Host)
  • Voting — 2–4 options, live results
  • Quiz buzzer — two teams race to buzz

Run the room

  • Wheel spinner — random picker
  • Countdown timer — synced to all screens
  • Host timer — just your screen
Moving between slots
  • 🎛 Controlled on the Host screen or phone
  • You decide when to move to the next slot
Designer shortcuts
  • Mirror on all 3 — set Host, copy everywhere (great for media)
  • Mirror TS1 → TS2 · Duplicate · drag to reorder
  • Drag & drop media onto the timeline · everything auto-saves
Keep a strong shape
  • Tag each slot with a phase:
  • Launch → Establish → Apply → Demonstrate
  • Start & end with a confidence check to show progress.
